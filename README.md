**VOID.rpc** — это мощный и легкий инструмент для полной кастомизации твоего статуса (Rich Presence) в Discord. Никакого лишнего геморроя, консольных окон и сложных конфигов — всё управляется через красивый графический интерфейс.

## 📥 Скачать готовый софт (Релизы)
Тебе не нужно устанавливать Python или собирать код самому. Просто скачай готовый автономный `.exe` файл:
👉 **[Скачать VOID.rpc v1.2](https://github.com/n0eS1337/VOID.rpc/releases/tag/VOID.rpc.1.2)**

## 🔥 Что нового в версии 1.2:
* 🗂 **Система пресетов:** Создавай, сохраняй и удаляй неограниченное количество кастомных статусов. Переключайся между ними в один клик.
* 🔄 **Цикл пресетов (Слайд-шоу):** Включи автоматическую ротацию статусов по таймеру, превратив свой профиль в живую ленту.
* 🛠 **Умная защита прав доступа:** Программа автоматически сбрасывает системные ограничения "Только для чтения" с файлов конфигурации. Больше никаких вылетов и вырвиглазных ошибок!
* 📊 **Контроль лимитов:** Встроенные интерактивные счётчики символов (до 128) загатятся красным, если ты превысишь ограничения Discord API.
* 📋 **Быстрые действия:** Добавлены кнопки мгновенной очистки полей (`×`) и копирования Application ID (`📋`) в буфер обмена.

---

## ⚡ Глобальные горячие клавиши (Hotkeys)
Программа отслеживает нажатия, даже если ты находишься в игре или приложение свернуто в трей:
* `Ctrl + Shift + H` — Быстро скрыть окно в системный трей или развернуть обратно.
* `Ctrl + Shift + X` — Мгновенно очистить текущий статус в Discord.

---

## 🚀 Основные возможности:
* **Гибкие таймеры:** Поддержка режимов «Идет вперед⏳», «Обратный отсчет⏱️» и «Заморожен🟥».
* **Кастомные кнопки:** Добавляй кликабельные ссылки (на свой Steam, Twitch или сайт) прямо в профиль игрового статуса.
* **Изображения без хлопот:** Встроенная загрузка картинок на хостинг в один клик.
* **Автозапуск:** Опция автоматического старта вместе с Windows через реестр без потери рабочих путей к иконкам.
* **Двуязычный интерфейс:** Полная локализация на Русский и Английский языки (RU/EN).
